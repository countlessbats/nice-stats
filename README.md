# Nice Stats

A tiny Pillars of Eternity 1 sidecar mod for people who looked at balanced progression and said
"what if the number were funnier?"

## Features

- Character creation gives new player characters and hired adventurers **69 attribute points**.
- Every primary party member gets a **Nice Stats** modal.
- The modal uses `ActivationGroup.None`, so it is designed to stack with normal modal abilities.
- While active, the modal applies temporary bonuses so Stealth, Athletics, Lore, Mechanics,
  Survival, and Crafting calculate to **69**.
- Turning the modal off removes only its own temporary bonuses.

The in-game activation message is the tooltip's spiritual understudy:
`Nice Stats: all skills are now 69. This is scholarship.`

## Installation

Close the game, then run `install.bat`. If the installer cannot find the game automatically, paste
the Pillars of Eternity install location when prompted.

## Internal Name

The hook identity is `LoomNiceStats`; the release DLL is `LoomNiceStats.dll`.
